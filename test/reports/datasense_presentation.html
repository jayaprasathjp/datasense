Brief about the idea

DataSense GPU is a natural-language data analytics agent that uses NVIDIA RAPIDS (cuDF + cuML) to deliver anomaly alerts and risk scoring up to 213x faster than traditional CPUs.

It solves the critical execution bottleneck for Retail Operations Managers and Support Leads handling massive datasets, turning minutes of wait time into sub-second instantaneous decisions.

02

How did you approach the problem?

  • 1. Isolated the Bottleneck: Benchmarked pure CPU (pandas/sklearn) vs GPU (cuDF/cuML) on up to 20M rows.
  • 2. AI Integration: Built a Gemma-4 E2B LLM pipeline with an autonomous self-healing execution loop.
  • 3. Real-World Data: Integrated actual eCommerce transactional data via Google BigQuery.
  • 4. The Result: Measured a massive 213x execution speedup on our core Risk Modeling task.
03

Opportunities & USP

The Problem:
Traditional LLM data agents run their generated Python code on standard CPUs. On datasets larger than 1 Million rows, this causes minutes-long delays, destroying the conversational UX.
Our USP:
DataSense combines the LLM's intelligent generation with instantaneous GPU execution (< 1s). The LLM thinks fast, and the GPU executes instantly, regardless of data scale.
